# Changelog

All notable changes to this repository will be documented in this file.

## [Unreleased]

- Initial conversion to ansible-role-dotmodules structure
- Constitution v1.0.0 with 8 core principles
- Governance model documentation
- Coding standards documentation
- Module structure: shell, git, editor, dev-tools
- Ansible playbooks for deployment

- Reorganized dotfiles into modular structure

- Direct Brewfile (packages now in module configs)
- Old directory structure (bin/, config/, dev/, git/, vim/)

### Added
- Initial release with modular dotfiles structure
