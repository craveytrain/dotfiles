<task type="auto">
  <name>Task 1: Deploy conf.d fragments via stow and run automated smoke tests</name>
  <files></files>
  <action>
Deploy the 4 modules that contribute fish conf.d fragments using GNU Stow. Follow the lessons from Phase 4 Plan 02:

**Pre-deployment cleanup:**
1. Check for and remove any stale symlinks in `~/.config/fish/conf.d/` that point to old/broken paths
2. Check for and remove any stale symlinks in `~/.config/fish/functions/` for mux.fish if it exists as a broken link

**Stow deployment:**
Deploy each module with `--no-folding` flag (ensures directory creation, not directory symlinks). Use `--ignore='config\.fish'` for editor, shell, and dev-tools modules to avoid cross-module config.fish conflicts (the mergeable files are still declared in config.yml until Phase 7).

```bash
cd ~/dotfiles  # or wherever the dotfiles repo root is
stow --no-folding -t ~ modules/fish/files
stow --no-folding --ignore='config\.fish' -t ~ modules/editor/files
stow --no-folding --ignore='config\.fish' -t ~ modules/shell/files
stow --no-folding --ignore='config\.fish' -t ~ modules/dev-tools/files
```

If stow fails due to existing files/symlinks conflicting, remove the conflicting targets first (they are old merged symlinks being replaced).

**Automated smoke tests** (run in fish shell via `fish -c`):
1. Verify 4 conf.d symlinks exist: `ls -la ~/.config/fish/conf.d/{10,50,80}-*.fish`
2. Verify mux.fish symlink exists: `readlink ~/.config/fish/functions/mux.fish`
3. Verify fish starts without errors: `fish -c "echo 'fish started ok'"` (exit code 0)
4. Verify env vars set: `fish -c "echo \$DOTFILES"` should output a path containing "dotfiles"
5. Verify EZA_COLORS set: `fish -c "echo \$EZA_COLORS"` should output `uu=33:un=33`
6. Verify mise available: `fish -c "which mise"` should succeed (PATH must be set correctly from 10-fish-core.fish)
7. Verify config.fish is minimal: `fish -c "cat ~/.config/fish/config.fish"` should show only the config.local.fish source block

Note: If fish is not available as a command (not in current shell PATH), locate it via `/opt/homebrew/bin/fish` or check brew installation.
